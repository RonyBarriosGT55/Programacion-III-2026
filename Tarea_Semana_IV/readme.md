## Descripción
El programa usa `struct Producto` y `struct Nodo` para construir manualmente una lista simplemente enlazada con 4 productos.

## Funcionamiento
1. Crea 4 nodos dinámicamente con `new`.
2. Enlaza los nodos manualmente en el orden: n1 -> n2 -> n3 -> n4 -> nullptr.
3. Solicita al usuario el código, nombre y precio de cada producto.
4. Recorre la lista con un puntero auxiliar e imprime los datos.
5. Calcula e imprime el total acumulado de precios.
6. Libera toda la memoria dinámica con `delete` antes de finalizar.
