Lista Enlazada de Productos

Este programa en C++ crea una lista enlazada simple para almacenar información de productos.

Qué hace el programa

Crea 3 nodos dinámicamente usando new.

Cada nodo guarda:

código del producto

nombre

precio

Los nodos se enlazan formando una lista.

El usuario ingresa los datos desde teclado.

El programa valida que código y precio sean números.

Muestra los productos ingresados.

Librea la memoria usando delete.

Cómo usarlo

Ejecutar el programa.

Ingresar los datos solicitados.

Si se escribe texto donde van números, mostrará:

No se permite ese valor de variable

Al final se muestra la lista de productos.

Ejemplo de datos

Codigo: 101
Nombre: Laptop
Precio: 5500.50
